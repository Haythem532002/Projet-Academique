CSS

CSS3 est le langage de style utilisé pour mettre en forme les pages web. Il permet de définir la présentation d'une page, en spécifiant la couleur, la police, la taille, la disposition et d'autres caractéristiques visuelles des éléments HTML.

Get Started

Principe des feuilles de style CSS

Les feuilles de style CSS permettent de séparer la présentation d'une page HTML de son contenu. Au lieu de définir les styles directement dans le code HTML, les développeurs web peuvent créer une feuille de style distincte qui contient toutes les règles de mise en forme. Cette approche permet de maintenir une structure claire et organisée du code HTML,de réduire la duplication de code et de faciliter les mises à jour de la mise en forme de la page.

Sélecteurs CSS class et id

Les sélecteurs CSS sont utilisés pour appliquer des styles à des éléments HTML spécifiques. Les classes et les identifiants sont deux types de sélecteurs qui permettent de cibler des éléments en fonction de leur attribut "class" ou "id". Les classes peuvent être utilisées pour regrouper des éléments qui ont des styles similaires, tandis que les identifiants sont utilisés pour cibler des éléments spécifiques dans une page.

Mettre en forme un texte en CSS

Le texte est l'un des éléments les plus couramment stylisés en CSS. Pour modifier l'apparence du texte, les développeurs peuvent utiliser des propriétés telles que "font-size", "font-family", "color", "text-align" et "text-decoration". Ces propriétés permettent de définir la taille de la police, la famille de la police, la couleur du texte, l'alignement et la décoration du texte. En utilisant ces propriétés, les développeurs peuvent créer des styles de texte personnalisés pour leur page web.

À Quoi Ressemble CSS ?

CSS est composé de selecteurs, proprietés, and valeurs.

Let's go over these by using an example. If you had an HTML paragraph that looked like this: <p>Hello world!</p> , you could style that paragraph with CSS.
It might look like this:

h1 {
color: white;
text-align: center;
}

This would change the font color to white and align it in the center of the page.
Note: You can also keep your CSS on one line, like so: p {font-family: verdana; font-size: 20px; color: blue}
